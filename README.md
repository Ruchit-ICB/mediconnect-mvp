# MEDICONNECT - Imagine Cup MVP

AI-powered Health Assistant for Rural & Underserved Communities.

## Features
- **Symptom Checker**: Select or type symptoms.
- **AI Analysis**: Mocked AI logic to determine risk (Low/Med/High).
- **Referrals**: Directs high-risk patients to nearest centers.
- **Multi-language Ready**: Language selection in profile (UI logic ready).

## Setup
1. `flutter pub get`
2. `flutter run`

## Directory Structure
- `lib/core`: Theme and constants.
- `lib/models`: Data models.
- `lib/providers`: State management.
- `lib/screens`: All UI screens.
- `lib/services`: Mock backend services.
- `lib/widgets`: Reusable UI components.
